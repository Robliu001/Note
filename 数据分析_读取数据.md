## 算术平均数
样板：S=[s1, s2, ... sn]
测量值 = 真值 + 误差
s1 = s + d1
s2 = s + d2 
...
sn = s + dn
算术平均数：m = (s1 + s2 + ... + sn)  / n = s+ (d1 + d2 + ... dn) / n 
n->oo, (d1 + d2 + ... dn) / n ->0, m-> s
用算数平均数作为真值的无偏估计
np.mean(S)->算术平均数

## 加权平均数
	每个样板对于真值所起的作用不一样，这样减少误差带来的影响。
	样本：S = [s1, s2, ... , sn]
	权重：W = [w1, w2, ... , wn]
	加权平均数：a = (s1w1 + s2w2 + ... snwn)/(w1 + w2 + ... wn)
	                         = s + (d1w1 + d2w2 + dnwn)/(w1 + w2 + ... wn)
	np.average(S, W)-> 加权平均数
